# 🎓 Campus Orbit

Campus Orbit is a modern college discovery and decision-making platform designed to help students find the right college, compare options, predict admission opportunities, and connect with a student community.

Built with a startup-grade user experience, Campus Orbit combines powerful search, admission prediction, college comparison, and community engagement into one platform.

---

## 🚀 Features

### 🔍 College Discovery

* Browse colleges across India
* Detailed college profiles
* Search and filter colleges
* Explore rankings, placements, fees, and courses

### ⚖️ College Comparison

* Compare multiple colleges side-by-side
* Evaluate placements, fees, rankings, and facilities
* Make informed admission decisions

### ⭐ Saved Colleges

* Save favorite colleges
* Build a personalized shortlist
* Access saved colleges anytime

### 🎯 Admission Predictor

* Predict college opportunities based on exam and rank
* Intelligent recommendation system
* Quick admission insights

### 💬 Community Discussions

* Ask questions
* Share experiences
* Answer student queries
* Discover trending discussions

### 👤 User Profiles

* Authentication system
* Personalized experience
* Profile management

## ── Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/campus-orbit.git
cd campus-orbit
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file:

```env
DATABASE_URL=your_database_url
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000
```

### Prisma Setup

```bash
npx prisma generate
npx prisma migrate dev
```

### Run Development Server

```bash
npm run dev
```

Visit:

```txt
http://localhost:3000
```
